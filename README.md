## requirement

- LINE BOT API Trial Account
- heroku account
- yelp account

## set up

- create an app in heroku
- set env
  - LINE_BOT_ID
  - LINE_BOT_SECRET
  - LINE_BOT_MID
  - YELP_KEY
  - YELP_SECRET
  - YELP_TOKEN
  - YELP_TOKEN_SECRET
- add 'Fixie' add-on
- set Fixie IPs in LINE whitelist
- set callback url in LINE BOT setting
- deploy this app to heroku
